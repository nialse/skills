# skills

This repo contains skills and skill development artifacts.

The repo is meta by design. It is for creating, reviewing, refining, and assessing skills rather than documenting the domain content of any single skill here.

## Purpose

- keep skills structured and reviewable
- keep claims grounded in primary sources where possible
- make authority order inside each skill explicit
- keep cross-references usable and coherent
- maintain a repeatable assessment standard

## Quality bar

Skills in this repo should aim to pass assessment with the highest scores across all factors.

That means they should be:

- complete for their stated scope
- fit for use by another agent without guesswork
- grounded in verifiable authority rather than unsupported taste
- explicitly sourced at the claim level when they make normative claims
- usefully cross-referenced with a clear authority order

## Assessment

The assessment rubric lives in [assessment.md](/Users/andreas/Documents/projekt/skills/assessment.md).

Use it as the shared standard when evaluating whether a skill is ready, what is missing, and what still needs revision.

## Validation

Skills in this repo should remain structurally valid under the relevant validation tooling for the skill format being used.

## Notes

- This repo favors narrow, testable skill improvements over broad unsourced guidance.
- External skill material should be normalized before import.
- Skill-specific content belongs inside the skill bundle, not in this top-level README.
