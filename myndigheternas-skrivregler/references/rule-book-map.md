# Rule to Book Map

This file maps skill rules to the books and is the fastest way to find the governing chapters.

## Working Order

- Omskrivning och granskning: börja med `R1` till `R5`, kontrollera sedan `R6` till `R11`.
- Ny text: börja med `R1`, `R3` och `R5`, kontrollera sedan `R2`, `R4` och `R6` till `R11`.
- Webbtext: använd `R1` till `R5` tillsammans med `R12`.
- Källhänvisning eller rättslig hänvisning: använd `R13` och komplettera vid behov med `R8`, `R10` eller `R11`.

## Response Pattern

- Ge förslag eller korrigering först och lägg motiveringen efteråt med `R*` och relevanta `D*` eller `S*`.
- Vid granskning redovisas först problem enligt `R1` till `R5`, därefter normfel enligt `R6` till `R11`.
