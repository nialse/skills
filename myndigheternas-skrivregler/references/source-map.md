# Source Map

Validated on 2026-03-15 against the official PDFs downloaded to `/tmp/myndigheternas-skrivregler-pdfs/`.

Use this file as the authority for all `D*`, `S*`, and `R*` references in `SKILL.md`. If a sentence in `SKILL.md` has no source IDs, treat it as procedural glue rather than a claim about the books.

## Verification Model

- `D*` means direct support. Each row supports only the claim written in that row and only within the cited chapter or page span.
- `S*` means explicit synthesis. Each synthesis row combines named `D*` rows into a reusable working rule and must not be read as a quotation from the books.
- Text in `SKILL.md` without `D*` or `S*` is procedural support: task order, response format, or other workflow glue.
- When a rule cites several `D*` rows, read them cumulatively. No single `D*` row is blanket support for every clause in the rule.
