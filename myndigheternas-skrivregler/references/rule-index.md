# Rule Index

Använd den här filen när du snabbt behöver välja regler per uppgift eller ämne. Läs inte [source-map.md](source-map.md) för vanlig omskrivning, bara när källstöd eller regelkontroll behövs.

## Uppgifter

| Uppgift | Börja med | Komplettera med |
| --- | --- | --- |
| Omskrivning till klarspråk | `R1`, `R2`, `R3`, `R5` | `R4`, `R6`-`R11` vid behov |
| Granskning med motivering | `R1`-`R5` | `R6`-`R11`; `R13` om texten har källor |
| Ny myndighetstext | `R1`, `R3`, `R5` | `R2`, `R4`, `R6`-`R11` |
| Webbtext eller länktext | `R1`, `R5`, `R12` | `R2`-`R4`, `R9` |
| Sociala medier | `R1`, `R3`, `R12` | `R4`, `R9`, `R11` |
| Normrättning | `R6`-`R11` | `R7` och `R8` vid namn, lagar och sammansättningar |
| Källhänvisning eller citat | `R13` | `R8`, `R10`, `R11` |
| Rättslig hänvisning | `R13` | `R8`, `R10`, `R11` |
