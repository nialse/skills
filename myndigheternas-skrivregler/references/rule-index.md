# Regelindex

Använd denna fil för att hitta rätt `R*` snabbt. För exakta bokkapitel, gå vidare till [rule-book-map.md](rule-book-map.md). För auktoritativa `D*` och `S*`, gå vidare till [source-map.md](source-map.md).

| Uppgift | Börja med | Fortsätt med | Aktivera bara när |

| Omskrivning | R1, R2, R3, R4, R5 | R6, R7, R8, R9, R10, R11 | R12 för skärmtext, R13 för källor eller rättsliga hänvisningar |
| Granskning | R1, R2, R3, R4, R5 | R6, R7, R8, R9, R10, R11 | R12 för skärmtext, R13 för källor eller rättsliga hänvisningar |
| Ny text | R1, R3, R5 | R2, R4, R6, R7, R8, R9, R10, R11 | R12 för skärmtext, R13 för källor eller rättsliga hänvisningar |
| Webbtext | R1, R2, R3, R4, R5, R12 | R11 | R13 bara när webbsidan också innehåller källor eller citat |
| Källhänvisning | R13 | R8, R10, R11 | R12 bara om materialet publiceras på webben |
